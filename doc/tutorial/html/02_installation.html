2. Installation

The only way to get HAL working right now is to compile its source code. But don't be afraid! It's pretty easy! HAL comes with all its dependencies, so you don't need to worry about them. To compile HAL you only need the Java Development Kit.

Once you have the JDK correctly installed, you just need to clone the repository in any directory:

git clone https://github.com/hecrj/hal
If you don't have git installed, you can still get the repository from here. Just unzip it wherever you want.

After you have your local copy of the repository ready, get inside that directory and run the Makefile to compile the source code:

cd hal
make

Once it's finished, you can find the HAL executable inside the folder bin. Thus, you can run HAL in interactive mode:

./bin/hal

Right now, just use Ctrl+D or type quit to exit the interactive shell.

If you want to be able to execute HAL from everywhere, then add an alias to your environment pointing to the bin/hal executable or include in your PATH the bin directory.

hal
